The `useResizeObserver()` hook listens to size changes on the given element and returns an object with `height` and `width`.
It uses the browser's `ResizeObserver` internally and cleans up the memory when necessary.

While the native `ResizeObserver` API differs between browsers and devices, this component provides a unified API by passing
an object with `width` and `height` consistently across all browsers and devices.

See also: <Link to='/docs/tools/resizeobserver'>&lt;ResizeObserver/&gt;</Link>

## Syntax

```js
const {width, height} = useResizeObserver(ref);
```

### Importing

```jsx
// ESM
import {useResizeObserver} from 'webrix/hooks';
import useResizeObserver from 'webrix/hooks/useResizeObserver'; // For better tree shaking

// UMD
const {useResizeObserver} = Webrix.hooks;
```

### Parameters

name|type|description
---|---|---
`ref`|componentRef|A React ref object (created using `useRef()`), pointing to the element to be observed.

### Return Value

Returns an object with height and width.
