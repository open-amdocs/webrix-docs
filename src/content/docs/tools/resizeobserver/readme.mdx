The `<ResizeObserver/>` can be used to listen to size changes in the given child component.
This component uses the browser's `ResizeObserver` internally, and cleans up the memory when necessary.

While the native `ResizeObserver` API differs between browsers and devices, this hook provides a unified API by returning
an object with `width` and `height` consistently across all browsers and devices.

See also: <Link to='/docs/hooks/useresizeobserver'>useResizeObserver()</Link>

## API

### Importing

```jsx
// ESM
import {ResizeObserver} from 'webrix/tools';
import ResizeObserver from 'webrix/tools/ResizeObserver'; // For better tree shaking

// UMD
const {ResizeObserver} = Webrix.tools;
```

### Props

Name|Type|Default|Description
---|---|---|---
`children`|node|`null`|Specifies a single child to observe for size changes.
`onResize`|function|`() => null`|Specifies an event handler to be called when the given child changes its size. The handler will receive an object with the keys `width` and `height`.
